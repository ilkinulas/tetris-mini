Tetris-Mini is an experiment project to learn kotlin-js.

Html5 Canvas is used to render the game. 

# Build
./gradlew clean build

# Run
open web/index.html


![Tetris-Mini Demo](gifs/tetris.gif)

# Controls
  Action     |     Key
  -----------|------------
  Move Left  | LEFT arrow
  Move Right | RIGHT arrow
  Rotate     | UP arrow
  Soft Drop  | DOWN arrow
  Hard Drop  | Space Bar
